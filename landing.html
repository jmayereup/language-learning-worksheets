Quick Integration

Integrate the worksheet component into any blog (Ghost, WordPress) or static site (Astro, Eleventy) with just a few lines of code.

1. Add Script & Styles

<script src="dist/wc/tj-pocketbase-worksheet.umd.js"></script>
<link rel="stylesheet" href="dist/wc/language-learning-worksheets.css">

2. Embed the Component

<tj-pocketbase-worksheet>
  <script type="application/json">
    { /* Raw PocketBase JSON Export */ }
  </script>
</tj-pocketbase-worksheet>
